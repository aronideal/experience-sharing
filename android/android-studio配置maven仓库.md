
## jcenter 是由 bintray.com 维护的 Maven 仓库 。

[jcenter()](http://jcenter.bintray.com)

```
allprojects {
    repositories {
        jcenter()
    }
}
```


## Maven Central 是由 sonatype.org 维护的 Maven 仓库。

[mavenCentral()](https://oss.sonatype.org/content/repositories/releases/)

```
allprojects {
    repositories {
        mavenCentral()
    }
}
```
